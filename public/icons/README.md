# CloudCrew Academy PWA Icons

This directory contains PWA icons for CloudCrew Academy. In a production environment, these would be actual icon files.

## Required Icons:
- favicon.ico (16x16, 32x32)
- icon-72x72.png
- icon-96x96.png
- icon-128x128.png
- icon-144x144.png
- icon-152x152.png
- icon-192x192.png
- icon-384x384.png
- icon-512x512.png
- apple-touch-icon.png (180x180)

## For Development:
The icons are referenced in the PWA manifest but not required for testing functionality.
You can generate these icons using tools like:
- PWA Builder
- Real Favicon Generator
- Icon generators

## Colors:
- Primary: #1e293b (slate-800)
- Secondary: #3b82f6 (blue-500)
- Accent: #8b5cf6 (purple-500)
